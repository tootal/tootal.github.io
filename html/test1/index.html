命令行操作

登录MySQL：先运行服务端程序mysqld,在另开一个窗口输入命令mysql -uroot -p
初始密码为空，直接回车即可。 重置root用户密码：若忘记了设置的密码，在my.ini文件末尾加上skip-grant-tables即可跳过密码直接登录。
然后输入：update mysql.user set authentication_string = password('my_new_password') where User = 'root' and Host = 'localhost'; flush privileges;
之后去掉skip-grant-tables重新登录即可。
